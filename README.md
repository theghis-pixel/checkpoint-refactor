# Pricing Card Refactorisée

## Objectif
Correction et refactorisation d’un composant pricing card avec IA.

## Ce qui a été corrigé
- Correction des erreurs HTML (balises mal fermées)
- Correction du CSS (box-shadow)
- Amélioration du layout
- Ajout d’un composant réutilisable PricingCard()
- Ajout d’un bouton interactif

## Structure
- before.html → version initiale avec bugs
- index.html → version corrigée et refactorisée

## Fonctionnalité principale
Composant JavaScript réutilisable :

PricingCard(title, price, features)

## Prompt IA utilisée
"Fix all HTML/CSS bugs in this pricing card, correct syntax errors, improve layout consistency, and refactor it into a reusable JavaScript component function PricingCard(title, price, features) while keeping it in a single HTML file."
